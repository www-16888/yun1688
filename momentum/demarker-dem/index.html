离散指标 (DeMarker - DeM) 深度解析

一、释义：它是什么？

核心概念

离散指标（DeMarker，简称DeM）由美国技术分析师托马斯·德马克（Thomas DeMark）于20世纪90年代提出，是一种基于“价格高低点相对变化”衡量市场多空动能强度的震荡指标。它通过对比“当期与前期价格的高低点差异”，计算多空动能的“相对离散程度”，最终将结果映射到0-1（或0%-100%）区间，核心逻辑是“当多头动能持续强于空头时，价格倾向于创新高，DeM值升高；当空头动能持续强于多头时，价格倾向于创新低，DeM值降低”——帮助交易者识别超买超卖状态、捕捉趋势反转信号，属于中灵敏度、侧重“动能离散度”的技术工具。

DeM的核心优势在于“聚焦价格高低点的主动变化，而非被动收盘价”——与RSI（基于收盘价涨跌幅度）、W%R（基于价格区间位置）不同，它通过“DeMax（多头动能）”和“DeMin（空头动能）”直接量化“市场主动推高/压低价格的力量”，对趋势初期的动能变化更敏感，尤其适合判断“趋势是否由动能驱动”。

  • 核心构成：
  • DeMax（多头动能因子）：若当期最高价 > 前期最高价，DeMax = 当期最高价 - 前期最高价；否则DeMax = 0（仅记录多头主动推高的幅度）；
  • DeMin（空头动能因子）：若当期最低价 < 前期最低价，DeMin = 前期最低价 - 当期最低价；否则DeMin = 0（仅记录空头主动压低的幅度）；
  • 指标计算逻辑：DeM = 短期SMA(DeMax) / [短期SMA(DeMax) + 短期SMA(DeMin)]，结果范围0-1（乘以100%后为0%-100%）；
  • 状态划分：
  • 超买区间：DeM > 0.7（或70%）——多头动能过度释放，价格上涨动力不足，大概率回调；
  • 超卖区间：DeM < 0.3（或30%）——空头动能过度释放，价格下跌动力不足，大概率反弹；
  • 中性区间：0.3 ≤ DeM ≤ 0.7——多空动能平衡，价格沿当前趋势运行，无明确反转信号；
  • 周期逻辑：默认参数为14期（最常用周期），周期越短，指标对短期动能变化越灵敏（如9期DeM易频繁进入超买超卖），周期越长，越能反映中长期动能趋势（如20期DeM信号更稳定）。

组成结构与计算逻辑

DeM的计算围绕“DeMax/DeMin动能因子”和“双因子SMA平滑”展开，核心参数为“计算周期（N）”（即SMA的平滑周期），默认14期，计算步骤分四步，逻辑清晰且可复现。

1. 核心计算步骤（以日线周期为例）

计算环节 计算逻辑 示例（周期N=14，当期数据：最高价108，最低价102；前期数据：最高价105，最低价103）
1. 计算DeMax（多头动能） 对比当期与前期最高价：
若当期最高价 > 前期最高价 → DeMax = 当期最高价 - 前期最高价
否则 → DeMax = 0		 当期最高价108 > 前期105 → DeMax = 108 - 105 = 3
2. 计算DeMin（空头动能） 对比当期与前期最低价：
若当期最低价 < 前期最低价 → DeMin = 前期最低价 - 当期最低价
否则 → DeMin = 0		 当期最低价102 < 前期103 → DeMin = 103 - 102 = 1
3. 计算双因子SMA 分别计算过去N期的SMA(DeMax)和SMA(DeMin)（简单移动平均）
——平滑短期动能波动，避免噪音		 假设过去14期SMA(DeMax)=2.5，SMA(DeMin)=1.5
4. 计算DeM值 DeM = SMA(DeMax) / [SMA(DeMax) + SMA(DeMin)]
（结果×100%后转化为百分比形式）		 DeM = 2.5 / (2.5 + 1.5) = 0.625 → 62.5%（中性区间）

2. 关键参数与周期适配

DeM的核心参数是“计算周期（N）”，不同周期对应不同的交易场景，需根据“交易风格”和“品种波动率”调整，避免信号滞后或噪音过多： - 短线交易（15分钟/1小时，日内期货/加密货币）：周期设为9-12期——短期周期灵敏度高，能快速捕捉日内“主动推高/压低”的动能变化（如1小时9期DeM，适合日内T+0捕捉1-2小时的反转机会）； - 中短线交易（4小时/日线，股票波段）：周期设为14期（默认）——14期覆盖2周左右行情，平衡灵敏度与稳定性，信号可靠性高，适合1-3周的波段交易； - 中长线交易（周线/月线，指数/大宗商品）：周期设为20-28期——长期周期过滤短期波动噪音，仅捕捉影响3个月以上的中长期动能趋势（如20期DeM突破70%，可能是中期顶部信号）； - 波动率适配：高波动品种（加密货币、原油期货）设短周期（9-14期），适配价格快速变化；低波动品种（蓝筹股、国债期货）设长周期（20-28期），减少“DeM频繁进出超买超卖区间”的假信号。

二、意义：它为什么有用？

DeM的核心价值在于用“主动动能因子”解决了“动能真假难辨”的痛点。传统震荡指标（如RSI）易受“被动收盘价波动”误导（如价格盘中冲高回落，收盘价仍涨，RSI显示多头强，但实际多头已乏力），而DeM通过“仅记录主动推高/压低的幅度”，精准识别“真实动能强度”，具体体现在三大不可替代的功能：

1. 精准识别“虚假动能”与“真实趋势”

DeM通过DeMax/DeMin的“非零比例”判断动能是否“主动”，避免被“被动价格波动”误导： - 真实多头趋势：价格上涨时，DeMax持续非零（多头主动推高），DeMin多为0（空头无主动压低），SMA(DeMax)远大于SMA(DeMin)，DeM值逐步升至70%附近——表明上涨由多头主动推动，趋势延续性强； - 虚假多头信号：价格上涨但DeMax频繁为0（无主动推高，仅被动跟随），DeMin偶尔非零（空头有主动压低），SMA(DeMax)与SMA(DeMin)差距小，DeM值在50%附近震荡——表明上涨是“惯性延续”，而非动能驱动，大概率回调； - 真实空头趋势：价格下跌时，DeMin持续非零（空头主动压低），DeMax多为0（多头无主动推高），SMA(DeMin)远大于SMA(DeMax)，DeM值逐步降至30%附近——表明下跌由空头主动推动，趋势延续性强； - 优势：相比RSI“只要收盘价涨就认为多头强”，DeM能区分“主动涨”和“被动涨”，在趋势初期的“动能验证”上准确率更高。

2. 提前捕捉趋势反转（比价格形态更灵敏）

DeM的“超买超卖突破”和“背离”是“趋势反转的强信号”，能在价格未出现明确反转形态时，提前1-3个周期提示动能衰竭： - 超买反转（看跌）：DeM突破70%超买区间后，逐步回落至70%以下，同时价格仍在创新高——表明多头主动推高的动能减弱，即使价格新高，也大概率回调（如DeM从75%降至65%，价格从110涨至112，反转概率超80%）； - 超卖反转（看涨）：DeM跌破30%超卖区间后，逐步回升至30%以上，同时价格仍在创新低——表明空头主动压低的动能减弱，即使价格新低，也大概率反弹（如DeM从25%升至35%，价格从90跌至88，反转概率超80%）； - 背离信号（强于超买超卖）： - 顶背离：价格创新高，DeM未同步创新高（如价格从100涨至105，DeM从72%降至68%）——表明“价格新高但多头主动动能衰减”，反转信号比单纯超买更可靠； - 底背离：价格创新低，DeM未同步创新低（如价格从100跌至95，DeM从28%升至32%）——表明“价格新低但空头主动动能衰减”，反转信号比单纯超卖更可靠；

3. 辅助过滤震荡市假信号（比趋势指标更精准）

DeM在震荡市中能通过“DeMax/DeMin的交替非零”判断“多空平衡”，帮助交易者避开“频繁追涨杀跌”的陷阱： - 震荡市确认：价格在固定区间内波动，DeMax和DeMin频繁交替非零（多头偶尔推高，空头偶尔压低），SMA(DeMax)与SMA(DeMin)数值接近，DeM值在0.3-0.7区间内来回震荡，无持续单边趋势； - 震荡市交易禁忌：震荡市中DeM的“超买超卖”多为假信号（如DeM短暂升至72%后立即回落，无真实多头动能），若按“超买卖、超卖买”交易，会导致“频繁止损”——需等待DeM突破0.7或跌破0.3后“站稳该区间”（如DeM连续3期>70%），确认动能方向后再入场； - 优势：相比MA（仅能判断趋势方向，无法区分震荡），DeM通过“动能离散度”直接标记震荡市，过滤假信号的效率更高。

三、用法：如何使用它？

DeM的用法围绕“动能验证”“超买超卖反转”“背离预警”三大场景展开，核心是“用DeMax/DeMin判断动能真假，用超买超卖+背离确认反转，用趋势过滤假信号”。

用法一：超买超卖反转交易（震荡市/趋势末期）

适用于震荡市或趋势末期动能衰竭场景，核心是“DeM突破超买/超卖区间后回落/回升时入场”，捕捉短期反转机会。

超卖反转买入（空头动能衰竭）

  • 反转信号确认（需满足3个条件）：
  • DeM跌破30%超卖区间（如DeM=25%），且持续至少1期——确认空头动能过度释放；
  • 随后DeM回升至30%以上（如从25%升至35%），且当期DeMax>0、DeMin=0——确认多头开始主动推高；
  • 价格未创新低（或出现“锤子线”等看涨K线）——价格与动能信号共振；
  • 入场时机：DeM回升至30%以上且收盘价站稳“前一期最高价”时入场；
  • 止损止盈：
  • 止损：DeM超卖区间的最低值对应的价格低点下方（如DeM=25%时价格低点90，止损设89.5）；
  • 止盈：DeM中性区间上沿（0.5-0.6）对应的价格高点（如历史震荡区间上沿100），或DeM接近70%时部分止盈。

超买反转卖出（多头动能衰竭）

  • 反转信号确认（需满足3个条件）：
  • DeM突破70%超买区间（如DeM=75%），且持续至少1期——确认多头动能过度释放；
  • 随后DeM回落至70%以下（如从75%降至65%），且当期DeMin>0、DeMax=0——确认空头开始主动压低；
  • 价格未创新高（或出现“流星线”等看跌K线）——价格与动能信号共振；
  • 入场时机：DeM回落至70%以下且收盘价跌破“前一期最低价”时入场；
  • 止损止盈：
  • 止损：DeM超买区间的最高值对应的价格高点上方（如DeM=75%时价格高点110，止损设110.5）；
  • 止盈：DeM中性区间下沿（0.4-0.5）对应的价格低点（如历史震荡区间下沿100），或DeM接近30%时部分止盈。

用法二：背离反转交易（趋势反转强信号）

适用于趋势末期（如上涨/下跌趋势持续一段时间后），核心是“价格与DeM背离时入场”，捕捉中期趋势反转，准确率高于单纯超买超卖。

底背离买入（下跌趋势反转）

  • 背离确认（需满足2个核心条件）：
  • 价格创新低：价格突破过去20期低点，创近期新低（如从100跌至95）；
  • DeM不创新低：DeM未同步突破过去20期低点，反而回升（如DeM从28%升至32%，未破前低25%）；
  • 强信号叠加（可选）：
  • 背离时，DeMin从非零转为0（空头无主动压低），DeMax首次出现非零（多头开始主动推高）；
  • 价格出现“双底”“头肩底”等看涨形态；
  • 入场时机：背离确认后，DeM回升至30%以上且价格突破“前一次反弹高点”时入场（右侧确认，降低左侧风险）；
  • 止损止盈：
  • 止损：价格新低下方（如95-1%=94.05），若价格重新破低，背离失效；
  • 止盈：前一波下跌的起点（如100），或DeM接近70%时部分止盈。

顶背离卖出（上涨趋势反转）

  • 背离确认（需满足2个核心条件）：
  • 价格创新高：价格突破过去20期高点，创近期新高（如从100涨至105）；
  • DeM不创新高：DeM未同步突破过去20期高点，反而回落（如DeM从72%降至68%，未破前高75%）；
  • 强信号叠加（可选）：
  • 背离时，DeMax从非零转为0（多头无主动推高），DeMin首次出现非零（空头开始主动压低）；
  • 价格出现“双顶”“头肩顶”等看跌形态；
  • 入场时机：背离确认后，DeM回落至70%以下且价格跌破“前一次回调低点”时入场；
  • 止损止盈：
  • 止损：价格新高上方（如105+1%=106.05），若价格重新破高，背离失效；
  • 止盈：前一波上涨的起点（如100），或DeM接近30%时部分止盈。

用法三：趋势跟踪交易（顺动能方向加仓）

适用于强劲单边趋势（配合ADX≥20确认趋势强度），核心是“顺DeM动能方向持仓，借DeMax/DeMin增强加仓”，赚足趋势主浪利润。

多头趋势跟踪（DeM>0.5持仓，DeMax增强加仓）

  • 趋势确认（需满足3个条件）：
  • DeM持续大于0.5（中性区间上沿），且逐步向70%靠近（如从0.5升至0.65）——多头动能主导；
  • DeMax持续非零（多头主动推高），DeMin多为0（空头无主动压低），SMA(DeMax)持续大于SMA(DeMin)——真实多头动能；
  • 价格在上升通道中（低点上移、高点上移），且ADX≥20（确认趋势强度）；
  • 加仓时机：
  • 第一次加仓：DeM突破前期高点（如从0.6升至0.65，破前高0.62），且当期DeMax>前一期DeMax（多头主动推高增强）时；
  • 第二次加仓：价格回调至MA20附近，DeM虽回落但仍>0.5，且DeMin=0（空头无主动打压）时；
  • 止损离场：
  • 止损：MA20下方（或DeM跌破0.5），若DeM回落到0.5以下，表明多头动能减弱；
  • 离场信号：DeM出现顶背离，或DeMax连续2期为0（多头无主动推高）。

空头趋势跟踪（DeM<0.5持仓，DeMin增强加仓）

  • 趋势确认（需满足3个条件）：
  • DeM持续小于0.5（中性区间下沿），且逐步向30%靠近（如从0.5降至0.35）——空头动能主导；
  • DeMin持续非零（空头主动压低），DeMax多为0（多头无主动推高），SMA(DeMin)持续大于SMA(DeMax)——真实空头动能；
  • 价格在下降通道中（高点下移、低点下移），且ADX≥20（确认趋势强度）；
  • 加仓时机：
  • 第一次加仓：DeM跌破前期低点（如从0.4降至0.35，破前低0.38），且当期DeMin>前一期DeMin（空头主动压低增强）时；
  • 第二次加仓：价格反弹至MA20附近，DeM虽回升但仍<0.5，且DeMax=0（多头无主动推高）时；
  • 止损离场：
  • 止损：MA20上方（或DeM突破0.5），若DeM回升到0.5以上，表明空头动能减弱；
  • 离场信号：DeM出现底背离，或DeMin连续2期为0（空头无主动压低）。

四、高级技巧与注意事项

1. 参数优化：适配不同品种与场景

DeM的默认参数（14期）需根据“交易周期”和“品种波动率”调整，核心是“周期越长，信号越稳定；周期越短，信号越灵敏”，具体适配如下： | 交易场景 | 推荐周期（N） | 适配品种 | 核心逻辑 | |----------------|---------------|-------------------------|-----------------------------------| | 日内短线（15min） | 9-10期 | 加密货币、原油期货 | 短周期捕捉日内“主动推高/压低”的动能变化，应对高波动 | | 日线波段 | 14期（默认） | 股票、黄金、商品期货 | 平衡灵敏度与稳定性，适配1-3周波段，信号可靠 | | 周线中长线 | 20-25期 | 指数、大宗商品（铜、原油）| 长周期过滤短期噪音，反映3-6个月的中期动能趋势 | | 低波动品种（蓝筹股） | 20-25期 | 茅台、苹果等大盘股 | 长周期减少“DeM频繁进出超买超卖区间”的假信号，避免频繁止损 |

2. 缺点与警示：避开这些“DeM陷阱”

DeM虽主动动能识别优势明显，但存在天生局限性，尤其在“极端行情”和“低波动市场”中易产生误导： - 极端行情（涨跌停）失真：在“一字涨跌停”中，价格高低点固定（如连续涨停，最高价=收盘价=最低价），导致DeMax=0（无更高价可推）、DeMin=0（无更低价比对），SMA(DeMax)和SMA(DeMin)同步回落，DeM值向0.5靠拢（中性）——无法反映真实强劲的多头/空头动能，此类行情中DeM失效，需结合成交量（涨停缩量=趋势延续，放量=分歧）判断； - 低波动市场假信号多：在低波动品种（如国债期货）中，价格高低点变化极小，DeMax和DeMin频繁为0，SMA(DeMax)与SMA(DeMin)接近0，导致DeM值在0.5附近震荡，偶尔因微小波动短暂突破30%/70%（如DeM=71%），实为假超买——需结合“成交量放大”判断，放量突破超买/超卖才可能是真实信号； - 震荡市中“超买超卖反复”：在宽幅震荡市中，DeM可能频繁突破30%/70%（如1个月内3次进入超买、2次进入超卖），若按“单次突破”交易，会导致“频繁止损”——需设置“突破后站稳N期”的条件（如DeM连续2期>70%才算有效超买），过滤假信号； - 单一信号不可信：DeM需与“价格形态（头肩顶、双底）”“成交量”结合——例如，DeM超卖但价格跌破趋势线，可能是“假反弹”；DeM顶背离但成交量放大，可能是“背离延续”（多头仍有资金入场）。

3. 常见误区：这些错误要避免

  • 误区1：忽略DeMax/DeMin的“主动性”，仅看DeM数值——认为“DeM>70%=超买”，忽略DeMax是否持续非零（如DeM=72%但DeMax连续3期=0，实为被动价格上涨，非主动动能）；正确做法是“看DeM数值的同时，检查DeMax/DeMin的非零比例，主动动能驱动的超买/超卖才有效”；
  • 误区2：震荡市中按“单次超买超卖”交易——在ADX<20的震荡市中，DeM单次突破30%/70%后易快速回落，此时交易易“追涨杀跌”；正确做法是“震荡市中等待DeM突破后站稳2期以上，且价格突破震荡区间，再入场”；
  • 误区3：背离出现就立即入场——认为“价格创新低+DeM不创新低=底背离”，忽略“DeM需回升至30%以上”的确认条件（如DeM从28%升至29%，未破30%，仍属空头动能主导）；正确做法是“背离后等待DeM突破30%/70%，或价格突破关键位，再入场，降低左侧风险”；
  • 误区4：用短期DeM判断长期趋势——用15分钟DeM判断日线趋势，导致“小周期动能信号干扰大周期判断”（如15分钟DeM顶背离，日线DeM仍在0.6，中期趋势未变）；正确做法是“大周期（日线）定方向（如日线DeM>0.5），小周期（1小时）找入场点（如1小时DeM超卖回升）”。

五、总结

DeM是技术分析中的“主动动能标尺”——它通过“DeMax/DeMin”量化市场“主动推高/压低”的力量，既能精准识别“虚假动能”与“真实趋势”，也能提前捕捉趋势反转（超买超卖+背离），还能过滤震荡市假信号，尤其适合“趋势初期动能验证”和“多空主动力量对比”。

但DeM并非“万能动能指标”，极端行情失真、低波动市场假信号多仍是其无法克服的局限。掌握DeM的核心，不在于“记住30%/70%的阈值”，而在于“结合DeMax/DeMin判断动能真假、用ADX区分趋势与震荡、用多工具验证信号”——始终以“主动动能是否持续”为核心，才能充分发挥其“精准识别动能”的优势，规避缺陷。
