负成交量指数（NVI）& 正成交量指数（PVI）指标深度解析：原理、计算与实战应用

负成交量指数（Negative Volume Index, NVI）与正成交量指数（Positive Volume Index, PVI）是由技术分析师保罗·戴萨特（Paul Dysart）在20世纪30年代提出，后经诺曼·福斯贝克（Norman Fosback）在《Stock Market Logic》中系统推广的量价联动型指标。二者核心逻辑围绕“成交量变化与资金行为的关联”展开，通过区分“缩量行情”与“放量行情”下的价格波动，判断市场中“聪明钱”（机构资金）与“散户资金”的主导权，最终服务于趋势判断、资金流向验证与反转预警。

一、核心理论基础：指标的底层逻辑假设

NVI与PVI的设计基于福斯贝克对“市场资金行为”的经典观察，这是理解两个指标的前提： 1. 缩量行情（NVI关注）：当成交量较前一日减少时，市场交易活跃度下降，此时价格波动更可能由“聪明钱”（机构、专业投资者）主导——因为机构资金布局通常低调，无需大量成交量推动价格，且倾向在散户交易清淡时入场/离场； 2. 放量行情（PVI关注）：当成交量较前一日增加时，市场交易活跃度上升，此时价格波动更可能由“散户资金”主导——因为散户交易往往伴随情绪驱动（如追涨杀跌），容易引发成交量骤增，而机构此时更可能“借散户情绪出货/吸筹”； 3. 趋势验证逻辑：长期趋势的形成需要“聪明钱”的持续推动，因此NVI的长期走势更能反映市场真实趋势；PVI则更多反映短期情绪波动，需结合NVI判断趋势的可持续性。

二、指标定义、计算逻辑与核心特性

（一）负成交量指数（NVI）：跟踪缩量行情的“聪明钱信号”

1. 核心定义

NVI是仅在“当日成交量＜前一日成交量”时更新，在放量时保持不变的累计型指标。其核心功能是通过缩量行情中的价格变化，捕捉机构资金的潜在布局轨迹，判断市场真实趋势方向。

2. 完整计算逻辑（以日线周期为例）

NVI的计算以“前一期NVI值”为基础，仅在缩量时调整，步骤如下： - 第一步：确定计算前提
对比当日成交量（Volₜ）与前一日成交量（Volₜ₋₁）：
- 若 Volₜ ＜ Volₜ₋₁（缩量）：计算当日NVI；
- 若 Volₜ ≥ Volₜ₋₁（放量/平量）：当日NVI = 前一日NVI（保持不变）。

  • 第二步：缩量时的NVI计算公式
    [ NVI_t = NVI_{t-1} + \left( \frac{C_t - C_{t-1}}{C_{t-1}} \right) \times NVI_{t-1} ]
    其中：
  • (NVI_t) = 当日NVI值；
  • (NVI_{t-1}) = 前一日NVI值（初始值通常设为1000，便于观察相对变化）；
  • (C_t) = 当日收盘价；

  • (C_{t-1}) = 前一日收盘价。

  • 第三步：公式简化理解
    缩量时的NVI变化本质是“前一日NVI值 × （1 + 当日价格涨跌幅）”，即通过价格涨跌幅对NVI进行累计调整，反映缩量行情中价格波动对“聪明钱趋势”的影响。

Day1 1000 20元 20.4元 2000手 1800手（＜2000） 缩量 1000 + (20.4-20)/20 ×1000 1020
Day2 1020 20.4元 20.2元 1800手 1900手（＞1800） 放量 保持前一日NVI 1020
Day3 1020 20.2元 20.6元 1900手 1700手（＜1900） 缩量 1020 + (20.6-20.2)/20.2 ×1020 ≈ 1020 + 20.2 1040.2

4. 核心特性

  • 缩量更新，放量停滞：仅在散户交易清淡时反映资金动向，避免短期情绪干扰；
  • 长期趋势导向：累计计算方式使其更擅长反映1-3个月以上的中期趋势，短期波动较小；
  • 无固定取值范围：数值随趋势累计增长/下降（如牛市中NVI从1000升至2000，熊市中降至500），需通过“与历史高点/低点对比”或“移动平均线”判断位置。

（二）正成交量指数（PVI）：跟踪放量行情的“散户情绪信号”

1. 核心定义

PVI是仅在“当日成交量＞前一日成交量”时更新，在缩量/平量时保持不变的累计型指标。其核心功能是通过放量行情中的价格变化，捕捉散户资金的情绪波动，辅助判断短期趋势的强弱与反转风险。

2. 完整计算逻辑（以日线周期为例）

PVI的计算逻辑与NVI对称，仅在放量时调整： - 第一步：确定计算前提
对比当日成交量（Volₜ）与前一日成交量（Volₜ₋₁）：
- 若 Volₜ ＞ Volₜ₋₁（放量）：计算当日PVI；
- 若 Volₜ ≤ Volₜ₋₁（缩量/平量）：当日PVI = 前一日PVI（保持不变）。

  • 第二步：放量时的PVI计算公式
    [ PVI_t = PVI_{t-1} + \left( \frac{C_t - C_{t-1}}{C_{t-1}} \right) \times PVI_{t-1} ]
    变量定义与NVI一致，初始值同样设为1000。

3. 计算示例（初始PVI=1000）

Day1 1000 20元 20.4元 2000手 2500手（＞2000） 放量 1000 + (20.4-20)/20 ×1000 1020
Day2 1020 20.4元 20.2元 2500手 2300手（＜2500） 缩量 保持前一日PVI 1020
Day3 1020 20.2元 20.6元 2300手 2800手（＞2300） 放量 1020 + (20.6-20.2)/20.2 ×1020 ≈ 1020 + 20.2 1040.2

4. 核心特性

  • 放量更新，缩量停滞：仅在散户情绪活跃时反映价格波动，直接关联短期交易热度；
  • 短期情绪导向：对成交量变化敏感，擅长反映1-2周内的短期趋势，波动较NVI更频繁；
  • 无固定取值范围：需结合“移动平均线”或“与NVI的联动关系”判断信号，单独使用易受情绪误导。

（三）NVI与PVI的核心差异对比

对比维度 负成交量指数（NVI） 正成交量指数（PVI）
更新条件 当日成交量＜前一日成交量（缩量） 当日成交量＞前一日成交量（放量）
主导资金 机构资金（聪明钱） 散户资金（情绪资金）
趋势周期 中期趋势（1-3个月以上） 短期趋势（1-2周内）
信号特点 稳定、滞后性弱（反映真实趋势） 灵敏、波动性强（反映情绪波动）
核心功能 判断长期趋势方向、验证趋势可持续性 捕捉短期情绪反转、预警趋势短期风险

三、实战应用：NVI与PVI的联动策略

NVI与PVI单独使用时信号价值有限（如NVI无法反映短期情绪，PVI易受散户噪音干扰），二者联动分析是实战核心，主要应用场景包括“趋势判断”“背离预警”“资金流向验证”三大类。

（一）场景1：趋势方向判断（结合移动平均线）

由于NVI和PVI均为累计型指标，需搭配“10日/20日移动平均线（MA）”过滤短期波动，明确趋势方向：

1. 多头趋势确认（适合建仓/持仓）

  • 核心条件（满足2项及以上）：
  • NVI ＞ NVI的20日MA（中期趋势由机构资金推动向上）；
  • PVI ＞ PVI的10日MA（短期散户情绪与机构趋势一致，无明显分歧）；
  • 价格持续在20日MA上方运行（价格趋势与量能趋势同步）；
  • 逻辑：机构主导的长期趋势向上，且散户情绪配合（放量时价格上涨），趋势可持续性强，适合持仓或在缩量回调时加仓。

2. 空头趋势确认（适合空仓/减仓）

  • 核心条件（满足2项及以上）：
  • NVI ＜ NVI的20日MA（中期趋势由机构资金推动向下）；
  • PVI ＜ PVI的10日MA（短期散户情绪与机构趋势一致，恐慌抛售）；
  • 价格持续在20日MA下方运行（价格趋势与量能趋势同步）；
  • 逻辑：机构主导的长期趋势向下，且散户情绪加剧下跌（放量时价格下跌），趋势风险高，适合空仓或在放量反弹时减仓。

3. 震荡趋势确认（适合观望/短线交易）

  • 核心条件：
  • NVI围绕其20日MA上下波动（机构资金无明确方向）；
  • PVI围绕其10日MA剧烈震荡（散户情绪反复）；
  • 逻辑：市场缺乏主导资金，趋势不明朗，需等待NVI突破MA后再入场，避免频繁交易。

（二）场景2：趋势反转预警（背离信号）

“价格与指标背离”是NVI/PVI最有效的反转信号，尤其当二者同时出现背离时，反转概率极高：

1. 顶部反转预警（适合止盈/做空）

  • NVI顶背离（核心信号）：
    条件：①价格创近期新高；②NVI未创同期新高，且NVI跌破其20日MA；
    逻辑：机构资金已在缩量时悄悄离场，价格新高缺乏真实资金支撑，随时可能回调。
  • PVI顶背离（辅助信号）：
    条件：①价格创近期新高；②PVI未创同期新高，且放量时价格涨幅收窄；
    逻辑：散户追涨情绪减弱，放量却无法推动价格创新高，短期顶部形成。
  • 实战操作：若NVI顶背离+PVI顶背离同时出现，立即减仓50%；若后续价格跌破20日MA，剩余仓位全部止盈。

2. 底部反转预警（适合建仓/做多）

  • NVI底背离（核心信号）：
    条件：①价格创近期新低；②NVI未创同期新低，且NVI突破其20日MA；
    逻辑：机构资金已在缩量时悄悄入场，价格新低是“诱空”，真实需求开始显现。
  • PVI底背离（辅助信号）：
    条件：①价格创近期新低；②PVI未创同期新低，且放量时价格跌幅收窄；
    逻辑：散户恐慌抛售情绪减弱，放量却无法推动价格创新低，短期底部形成。
  • 实战操作：若NVI底背离+PVI底背离同时出现，轻仓建仓（仓位20%）；若后续NVI持续在MA上方，且价格突破20日MA，加仓至50%。

（三）场景3：资金流向验证（判断趋势真实性）

当价格趋势与NVI/PVI出现“分歧”时，需通过指标验证趋势是否由“真实资金”推动，避免陷入“假趋势”陷阱：

1. 验证“价格上涨趋势”的真实性

  • 真实上涨（可持仓）：价格上涨 + NVI上涨 + PVI上涨；
    逻辑：机构（缩量时推涨）与散户（放量时追涨）方向一致，趋势有资金支撑。
  • 假上涨（需警惕）：价格上涨 + NVI下跌 + PVI上涨；
    逻辑：仅散户放量追涨，机构在缩量时悄悄出货，上涨是“情绪驱动”，随时可能回调（典型“量价背离”的延伸）。

2. 验证“价格下跌趋势”的真实性

  • 真实下跌（可空仓）：价格下跌 + NVI下跌 + PVI下跌；
    逻辑：机构（缩量时抛售）与散户（放量时恐慌）方向一致，趋势有资金支撑。
  • 假下跌（可观望）：价格下跌 + NVI上涨 + PVI下跌；
    逻辑：仅散户放量抛售，机构在缩量时悄悄吸筹，下跌是“情绪恐慌”，随时可能反弹（典型“黄金坑”前兆）。

四、关键注意事项与常见误区

1. 避免“单一指标依赖”

NVI/PVI的核心价值在于“联动分析”，单独使用易出错：
- 仅看NVI：无法捕捉短期情绪反转（如NVI长期上涨，但PVI突然顶背离，短期仍需减仓）；
- 仅看PVI：易被散户情绪误导（如PVI短期上涨，但NVI下跌，长期趋势仍向下）。

2. 适配“高流动性品种”

NVI/PVI依赖“成交量与价格的连续联动”，低流动性品种（如小盘股、冷门ETF）因“成交量骤增骤减”或“价格跳空”，易导致指标计算失真：
- 适配品种：大盘股（如沪深300成分股）、主力合约期货、宽基指数（如上证指数、标普500）；
- 规避品种：日成交量＜5000手的小盘股、单日涨跌幅＞10%且成交量异常的股票。

3. 周期选择需匹配交易目标

  • 短线交易（1-3天）：用“日线PVI+10日MA”捕捉情绪反转，NVI仅作参考；
  • 波段交易（3-30天）：用“日线NVI+20日MA + 日线PVI+10日MA”联动，兼顾趋势与情绪；
  • 中长线交易（1-6个月）：用“周线NVI+20周MA”判断长期趋势，忽略日线PVI的短期波动。

4. 警惕“极端行情失效”

在“一字涨跌停”“停牌复牌”等极端行情中，成交量与价格的联动关系被破坏，指标信号无效：
- 一字涨停：成交量骤减（散户无法入场），NVI可能持续上涨，但价格已封死，无交易机会；
- 停牌复牌：复牌后成交量骤增（散户集中交易），PVI剧烈波动，但趋势缺乏连续性，需等待1-2个交易日再判断。

五、总结

NVI与PVI是一对“互补性极强的量价指标”——NVI像“市场的冷静观察者”，通过缩量行情捕捉机构资金的真实意图，反映长期趋势；PVI像“市场的情绪探测器”，通过放量行情捕捉散户资金的情绪波动，预警短期风险。

实战中，需牢记“NVI定方向，PVI看情绪，MA过滤噪音”的核心逻辑：用NVI判断长期趋势是否值得参与，用PVI寻找短期入场/离场时机，用移动平均线减少信号误判。最终，结合价格形态（如头肩顶、双底）与其他量能指标（如成交量均线），形成多维度验证，才能最大化NVI与PVI的实战价值，在趋势中把握机会，在反转前规避风险。
